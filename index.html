Project 3: Adaptive Prosthetic Control via Hybrid Learning

Introduction & Social Value: Lower extremity mobility challenges affect millions—150,000 transtibial amputations occur annually in the U.S. This work develops adaptive controllers for ankle prostheses to restore natural gait and improve users’ independence.

IMRD Format

Methods: Designed a hybrid pipeline combining Variational Adversarial Imitation Learning (VAIL) and Proximal Policy Optimization (PPO) in the MuJoCo simulation. The VAIL agent bootstraps expert gait from demonstration data; the PPO-based MLP agent refines ankle control for environmental adaptation. Implemented in Python with Gym and Stable Baselines3.

Results:

  • Achieved 43,088 avg. steps per rollout (MLP, 22 states) with MAE 10.17 Nm.
  • Hybrid approach outperformed pure IL (limited adaptation) and pure RL (unstable gait).
  • MLP-based controller matched transformer variants while using 80% fewer parameters.

Discussion: The hybrid pipeline effectively balances expert-like consistency with adaptive robustness. Reducing state inputs maintained performance, suggesting efficient feature selection. Next steps include testing with real hardware and integrating proprioceptive feedback.

My Role & Skills: I architected the learning framework, implemented MuJoCo environments, and tuned PPO hyperparameters. Key skills: Python, Reinforcement Learning (PPO, VAIL), MuJoCo, deep learning, environment integration.
